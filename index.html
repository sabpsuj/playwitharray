forEach()

I'm sure that you know the for loop. Sometimes it's a bit tricky to use, and actually, that is not the most elegant way to loop over the array. forEach() method comes here to handle it and simplify your code. Just use it instead of your loop. Callback for this inbuilt method is another function, which is executed once for each element while the forEach method is looping over your array.

Feel free to use this snippet and see the output in your console. As you can see we've looped through our array and we consoled each element with its index and current value. So, as you can see the general usage of this method is just handling the execution of another function on each element of an array.

And below another example:

The first line, we declared the variable called food. Then, below you can see the fat arrow function which concat v with string (here, as you can see it's a single letter s). It's an anonymous function stored in multi variable, below we are using it as a callback to our forEach() method. And now you know how to multiply your food resources in a simple way ;).

forEach() method returns undefined.

map()

Next method I'd like to introduce you is a map() method. It's a bit different from the previous one, but as a standard feature, you can find here also a loop over an array. Look at my little portion of code and see how it works. As previously I declared a variable which stores a function. This time I used another JavaScript built-in function which is available for a Math object. Math.pow allows us to return the base to the exponent power. First parameter here, is responsible for each element in the array, and the second one indicates the power value - here we have 2.

Then you can see that I declared another variable to assign a value from our map function. It's callback, as previously, is declared before function. The output of this method will be a totally new array with values changed with the pattern function (square).

map() method returns new array.
